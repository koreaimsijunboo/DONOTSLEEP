## WHAT IS THIS

DONOTSLEEP is an AI-based sleep detection system that uses your webcam to detect eye closure in real time.

<sub>DONOTSLEEP은 웹캠을 활용하여 눈 감김 상태를 감지하는 AI 기반 졸음 감지 시스템입니다.</sub>

---

## FEATURES

- Eye tracking using EAR algorithm  
- Real-time sleep detection timer  
- Alarm trigger system  
- Automatic image capture  
- Server-based storage (no localStorage)  
- Background music control  

<sub>
- EAR 알고리즘 기반 눈 추적  
- 실시간 졸음 감지 타이머  
- 알람 시스템  
- 자동 캡처 기능  
- 서버 저장 방식  
- 배경음악 제어
</sub>
